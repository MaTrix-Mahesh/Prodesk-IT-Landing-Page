# 🚀 Prodesk IT - Digital Agency Website

A modern, responsive landing page for a digital agency **Prodesk IT** built using **HTML, CSS, and JavaScript**.

## ✨ Features

- 🌐 Responsive Design (Mobile + Desktop)
- 🎨 Modern UI (Glassmorphism + Gradient Design)
- 🌙 Dark / Light Mode Toggle
- 📱 Mobile Navigation (Hamburger Menu)
- ⚡ Smooth Animations & Hover Effects
- 🧩 Clean and Structured Code

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla JS)

---

## 📸 Sections Included

- Navbar (Sticky + Responsive)
- Hero Section
- Services Section
- Footer

- Portfolio showcase
